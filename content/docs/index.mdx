**asciicn** is a monochrome component library distributed as a shadcn registry.
Components live in `components/asciicn` and can be installed with the shadcn CLI.

## What you get

- **Docs** powered by [Fumadocs](https://fumadocs.dev)
- **Components** with live previews and install commands
- **Registry** JSON built from source on every `dev` / `build`
