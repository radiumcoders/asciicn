**asciicn** is a monochrome component registry. Components live in
`components/asciicn`, are built into registry JSON, and can be installed with
the shadcn CLI.

## What you get

- **Docs** powered by [Fumadocs](https://fumadocs.dev)
- **Registry** built from `components/asciicn` on every `dev` / `build`
- **Install** via namespaced shadcn add commands

## Quick start

```bash
# Install a component from this registry (after you host it)
pnpm dlx shadcn@latest add @asciicn/ascii-badge
```

Local development:

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000/docs](http://localhost:3000/docs).
