Components in this registry are authored under `components/asciicn` and
published as built JSON under `/r`.

## Ascii Badge

Monochrome badge used as a sample registry item.

```bash
pnpm dlx shadcn@latest add @asciicn/ascii-badge
```

```tsx
import { AsciiBadge } from "@/components/asciicn/ascii-badge"

export function Example() {
  return (
    <>
      <AsciiBadge>default</AsciiBadge>
      <AsciiBadge variant="outline">outline</AsciiBadge>
    </>
  )
}
```
